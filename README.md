# weather-camera

![weather](picture.jpg)

Usage
=====
For a raspberrypi supporting it, I just made a call via crontab to run
every 5 minutes.

Weather
=======
It was extendend to use as script to gather weather information via
https://openweathermap.org/ API.

============
